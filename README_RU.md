# Sunny English - GitHub Pages

Эта папка уже подготовлена для публикации сайта через GitHub Pages.

## Как открыть сайт через GitHub

1. Создайте новый репозиторий на GitHub, например `sunnyenglish`.
2. Загрузите в репозиторий все файлы из этой папки.
3. Откройте `Settings` -> `Pages`.
4. В разделе `Build and deployment` выберите:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main`
   - папка: `/ (root)`
5. Нажмите `Save`.

Обычно через несколько минут, иногда до 10 минут, сайт будет доступен по адресу:

```text
https://ВАШ-ЛОГИН.github.io/sunnyenglish/
```

Главная страница называется `index.html`, поэтому GitHub Pages откроет ее автоматически.
